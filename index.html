🤖 Original AI Tracker (alex-jb/spacex-ipo-tracker)

This page extends the original spacex-ipo-tracker project by Alex Ji — a daily AI-managed stock research system on SpaceX IPO beneficiaries. The original tracker runs a multi-agent LLM (Bull/Bear/Judge) with 9 ML technical signals, publishing daily briefs since 2026-05-12.

How the daily brief is produced

  • Ranked BUY/WAIT/SELL — Orallexa multi-agent LLM (Bull / Bear / Judge / Critic / Memory)
  • Setup classifier — Trend / Mean-reversion bounce / Breakdown / Trend-chase warning
  • Position sizing tier — Full / Half / Tiny / Short-half / Pass (1-2% account risk)
  • FTD detector — Follow-Through Day on S&P 500 and NASDAQ
  • VIX regime — Calm / mildly elevated / elevated / panic
  • Brier audit (nightly) — Every BUY/SELL judged against next-day close

The honest part

This tracker is not investment advice. It runs in paper-trade mode. Every BUY/SELL/WAIT decision is committed to git the same day, then Brier-audited against ground truth. If the AI is full of it, you'll see it in the audit.
